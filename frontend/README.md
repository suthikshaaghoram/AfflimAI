# AffirmAI - AI-Powered Manifestation & Affirmations

Create personalized manifestations and affirmations powered by AI. Transform your goals into reality with custom Tamil-accent audio and guided visualization.

## Project info

This project is built with:
- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

To get started with development:

1. Clone the repository
```sh
git clone <YOUR_GIT_URL>
```

2. Install dependencies
```sh
npm install
# or
bun install
```

3. Start the development server
```sh
npm run dev
# or
bun dev
```

## Deployment

Build the project for production:

```sh
npm run build
```
